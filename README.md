## Description

Just a cookbook.

## Api

The Meal DB data structure:

### categories:

```json
{
  "categories": [
    {
      "idCategory": "1",
      "strCategory": "Beef",
      "strCategoryThumb": "https://www.themealdb.com",
      "strCategoryDescription": "Beef is the culinary...."
    }
  ]
}
```

### meals:

```json
{
  "meals": [
    {
      "strMeal": "Baked salmon with fennel & tomatoes",
      "strMealThumb": "https://www.themealdb.com",
      "idMeal": "52874"
    }
  ]
}
```
